Quadruped

    Google drive with images
    When:  2015 - 2018
    What:  12 Degrees of Freedom Quadrupedal Robot
Description:

The most recent (but not final) version of the quadruped is finished up untill the purchase of the servos. (They got expensive)

Its body was designed in CAD software and then 3D printed to fit some old servos found lying around.

An arduino was integrated as a servo controller in which a Raspberry Pi would send commands via serial to move each servo. The movement was calculated using inverse kinematics and some ultrasonic sensors that would work as a rudimentary obstacle avoidance.

The project was started as a way to begin learning about programming and robotics.
